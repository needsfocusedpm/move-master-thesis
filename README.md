# MOVE Master Thesis

This repository contains the structured public version of Daniel Hendling’s Master's Thesis on needs-focused leadership in project management and the MOVE Model.

This thesis is also published under the extended title:
"Needs-Focused Leadership in Projects: The MOVE Model on Motivation, Mediation, and Trust"

## Purpose

This repository provides:

- a canonical structured source of the thesis  
- machine-readable Markdown files for AI-supported use  
- a chapter-based structure for research and citation  
- a stable public access point alongside the PDF version  

## Structure

- `index.md` – central entry point  
- `chapters/` – structured thesis content  
- `pdf/` – formatted PDF version  
- `metadata/` – additional reference information  
- `CITATION.cff` – citation metadata
